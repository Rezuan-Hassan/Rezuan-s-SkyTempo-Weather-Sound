# 🌦️ Rezuan's SkyTempo: Weather & Sound 🎵

Welcome to **Rezuan's SkyTempo**, a premium, all-in-one dynamic dashboard that blends real-time weather tracking with your favorite music. Designed with a sleek, modern **glassmorphism** theme, this website provides a visually stunning experience for staying updated and entertained.

## ✨ Key Features
- **Global Weather Forecast:** Real-time temperature, wind speed, humidity, and precipitation data for any city worldwide using the OpenWeatherMap API.
- **Dynamic Backgrounds:** The UI theme changes based on the weather (e.g., stormy clouds for rain, bright fields for sunny days).
- **Integrated Music Player:** Paste any YouTube playlist link and listen to your "mood beats" directly within the dashboard.
- **Glassmorphism Design:** A beautiful, translucent UI inspired by modern desktop application aesthetics.
- **Responsive Layout:** Works smoothly across different screen sizes.
